# Snake Game

In this game... Well, snake game is so popular. I don't think I need to introduce the rules.

To compile and play, simply run the following commands:
```bash
> cd src
> javac -cp ".:../lib/stdlib.jar" *.java
> java -cp ".:../lib/stdlib.jar" Demo
```
